
# MkDocs (mkdocs)

MkDocs is a fast, simple and downright gorgeous static site generator that's geared towards building project documentation.

## Example Usage

```json
"features": {
    "ghcr.io/devcontainers-contrib/features/mkdocs:1": {}
}
```
